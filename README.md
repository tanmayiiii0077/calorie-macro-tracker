# Calorie Macro Tracker

## Features
- Track calories and macronutrients effortlessly.
- User-friendly interface for adding and editing food items.
- Searchable food database with nutritional information.
- Track meals and visualize nutritional intake.

## Usage Guide
1. **Installation**
   - Clone the repository: `git clone https://github.com/tanmayiiii0077hii/calorie-macro-tracker`
   - Navigate to the directory: `cd calorie-macro-tracker`
   - Install dependencies: `npm install`

2. **Running the Application**
   - Start the application: `npm start`
   - Open your browser and go to `http://localhost:3000`

3. **Adding Foods**
   - Click on the "Add Food" button.
   - Fill in the nutritional details and save.

## Deployment Instructions
- Ensure that your environment has Node.js and npm installed.
- Use the command `npm run build` to prepare the application for production.
- Deploy the contents of the `build` directory to your web server.

## Food Database Details
- The food database is integrated with a third-party API that provides real-time nutritional information.
- Suitable for various dietary preferences including vegetarian, vegan, and keto.

## Troubleshooting
- **Issue:** Application won't start.
  - **Solution:** Check for missing dependencies and ensure that you are in the correct directory.

- **Issue:** Food item not found in the database.
  - **Solution:** Add the item manually using the "Add Food" feature.

## Keyboard Shortcuts
- **Ctrl + N:** Add a new food item.
- **Ctrl + H:** Open the help menu.
- **Ctrl + S:** Save current meal plan.
